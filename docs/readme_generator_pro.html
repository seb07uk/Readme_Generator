Requirements & Installation

Up and running
in 60 seconds

System Requirements

Python 3.10+
Required for match/case syntax and union type hints
customtkinter ≥ 5.2
The only external dependency — modern Tk widgets
Windows / macOS / Linux
Any OS with Tk support (includes most Linux distros)
Git (optional)
Required only for Git integration features — must be in PATH
256 MB RAM
Minimum free memory for comfortable operation

Built-in stdlib (no pip)

os, sys, json, pathlib
File system operations and JSON config handling
threading
Background analysis and Watch mode threads
subprocess
Git command invocations with 5s timeout
ast
Python source code parsing for metadata extraction
hashlib, re, time, datetime
Watch mode checksums, regex, timestamps
1

Install dependency

pip install customtkinter

That's the only external package you need.

2

Launch GUI

python readme_generator_app.py

The app opens in dark mode with the standalone backend.

3

Optional: full backend

cp readme_generator.py ./

Place alongside the app for advanced plugin features.

4

Optional: CLI mode

python readme_generator_app.py --cli /path

Headless generation for hooks and pipelines.

Full backend mode: If readme_generator.py is present in the same directory, the application automatically loads it via importlib.util, replacing all fallback classes with the full-featured implementations. Status is shown in the status bar on startup.
