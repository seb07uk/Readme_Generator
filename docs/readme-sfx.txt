───────────────────────────────────────
  WHAT IS THIS?
───────────────────────────────────────

  A module that automatically analyzes a software project
  and generates a ready-to-use README.md file.

  Supported languages: Python, JavaScript, TypeScript, C#,
  Java, Go, Rust, C++, Ruby, PHP, Kotlin, Swift and more.

───────────────────────────────────────
  MAIN FEATURES
───────────────────────────────────────

  [+] Project structure analysis (files, directories, size)
  [+] Programming language detection and statistics
  [+] Metadata extraction: author, version, description,
      classes, functions
  [+] Git repository analysis (branch, commits, remote URL)
  [+] Test framework detection + run command
  [+] Documentation quality report (missing docstrings)
  [+] shields.io badge generation
  [+] Automatic Changelog from git log or CHANGELOG.md
  [+] Usage examples generated from public functions
  [+] Watch Mode – auto-regeneration on file changes
  [+] API documentation with types (AST type hint analysis)
  [+] Directory tree: compact or full mode
  [+] Project configuration via readmegen.json

───────────────────────────────────────
  WHAT'S NEW IN v2.1

───────────────────────────────────────
  GRAPHICAL INTERFACE (GUI)
───────────────────────────────────────

  Tabbed panel with the following sections:
    • Analysis    – files, languages, dependencies overview
    • Quality     – docstring report, percentage score
    • Git         – branch, commits, repository status
    • Settings    – templates, badges, README sections

  Action buttons:
    [Generate README]   [Generate Docs]   [Snapshot API]
    [Install Hook]      [Generate Changelog]

───────────────────────────────────────
  REQUIREMENTS
───────────────────────────────────────

  • Python 3.8 or newer
  • Libraries: tkinter (built-in), pathlib, ast, subprocess
  • Base module: main.py (BaseModule) – required in project
  • Optional: Git repository for commit analysis
