## Requirements

- Python 3.10+
- `customtkinter >= 5.2`

```bash
pip install customtkinter
```

---

## Running

```bash
# GUI
python readme_generator_app.py

# CLI (without GUI, e.g. as a pre-commit hook)
python readme_generator_app.py --cli /path/to/project
```

---

## Key Features

- 🔍 Project analysis — language, files, classes, functions, dependencies, Git
- 🎨 README templates — Minimalist, Open-Source, Enterprise, polsoft.ITS™
- 📑 Section selection — enable/disable any README sections
- 🪝 Pre-commit hook — auto-regeneration before each `git commit`
- 📸 Snapshot API — breaking change detection
- 📜 Changelog — from Conventional Commits
- 🌍 UI language — EN 🇬🇧 / PL 🇵🇱
